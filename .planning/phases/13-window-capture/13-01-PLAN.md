    3. **`buildCaptureScript(titleQuery: string, outputPath: string): string`** — exported for testability. Returns a PowerShell script string containing:
       - `Add-Type -AssemblyName System.Drawing`
       - `Add-Type -TypeDefinition @"..."@` with a C# class `WinCapture` containing:
         - P/Invoke imports: `EnumWindows`, `IsWindowVisible`, `GetWindowText`, `PrintWindow`, `DwmGetWindowAttribute`, `SetProcessDpiAwarenessContext`, `IsIconic`, `GetWindowPlacement`
         - `RECT` struct: `{ int Left, Top, Right, Bottom; }`
         - `WINDOWPLACEMENT` struct: `{ int length; int flags; int showCmd; POINT ptMinPosition; POINT ptMaxPosition; RECT rcNormalPosition; }` with `POINT` struct `{ int X, Y; }`
         - Constants: `DWMWA_EXTENDED_FRAME_BOUNDS = 9`, `PW_RENDERFULLCONTENT = 0x2`
         - `CaptureWindow(string titleQuery, string outputPath)` method:
           a. First call: `SetProcessDpiAwarenessContext(new IntPtr(-4))` (PER_MONITOR_AWARE_V2)
           b. `EnumWindows` delegate: `IsWindowVisible` check, `GetWindowText` check, case-insensitive substring match via `IndexOf(titleQuery, StringComparison.OrdinalIgnoreCase)`, stop on first match
           c. If no match: return `"ERROR:NO_MATCH"`
           d. Check `IsIconic(target)` for minimized window:
              - If minimized: call `GetWindowPlacement` to get `rcNormalPosition`, use those dimensions
              - If not minimized: call `DwmGetWindowAttribute(target, DWMWA_EXTENDED_FRAME_BOUNDS, out bounds, Marshal.SizeOf(typeof(RECT)))` for physical pixel bounds
           e. Validate `width > 0 && height > 0` — if not, return `"ERROR:ZERO_BOUNDS"`
           f. Create `Bitmap(width, height)`, get HDC via `Graphics.FromImage(bmp).GetHdc()`
           g. Call `PrintWindow(target, hdc, PW_RENDERFULLCONTENT)` — if returns false, return `"ERROR:PRINTWINDOW_FAILED"`
           h. `ReleaseHdc(hdc)`, `bmp.Save(outputPath, ImageFormat.Png)`
           i. Return `"OK:" + outputPath`
           j. Wrap entire method body in try/catch — catch returns `"ERROR:" + ex.Message`
       - PS invocation: `$result = [WinCapture]::CaptureWindow("${safeTitle}", "${safePath}")` then `Write-Output $result`
       - Title sanitization: `titleQuery.replace(/["`\r\n]/g, '')` before interpolation
       - Path escaping: `outputPath.replace(/\\/g, '\\\\')`

    4. **`captureWindow(titleQuery: string): CaptureResult`** — exported:
       - Generate `outputPath = path.join(SCREENSHOT_DIR, crypto.randomUUID() + '.png')`
       - `fs.mkdirSync(SCREENSHOT_DIR, { recursive: true })`
       - Call `spawnSync('powershell.exe', ['-NoProfile', '-NonInteractive', '-Command', buildCaptureScript(titleQuery, outputPath)], { encoding: 'utf8', timeout: 15_000 })`
       - If `result.error`: return `{ ok: false, error: \`spawn error: \${result.error.message}\` }`
       - If `result.status !== 0`: return `{ ok: false, error: \`PS exited \${result.status}: \${result.stderr?.trim()}\` }`
       - Parse stdout: `OK:` prefix -> `{ ok: true, path: stdout.slice(3) }`, `ERROR:` prefix -> `{ ok: false, error: stdout.slice(6) }`, else `{ ok: false, error: \`unexpected output: \${stdout}\` }`
